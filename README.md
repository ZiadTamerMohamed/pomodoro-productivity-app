# ⏱️ Pomodoro Timer Application

تطبيق مؤقت البومودورو (Pomodoro Timer) بسيط وأنيق مبني باستخدام تقنيات الويب الأساسية (HTML5, CSS3, JavaScript). يساعدك هذا التطبيق على تنظيم وقتك وزيادة إنتاجيتك من خلال تقسيم فترات العمل والراحة.

A simple, elegant, and responsive Pomodoro Timer web application built with vanilla HTML5, CSS3, and JavaScript. This tool is designed to help users manage their time effectively and boost productivity using the classic Pomodoro technique.

---

##  المميزات | Features

* **ثلاثة أنظمة للوقت:** التبديل السهل بين فترات العمل (Pomodoro)، الاستراحة القصيرة (Short Break)، والاستراحة الطويلة (Long Break).
* **واجهة مستخدم تفاعلية:** تصميم عصري ومريح للعين مع عداد دائري (SVG Circle Animation) يتفاعل مع الوقت.
* **التحكم الذكي:** إمكانية تشغيل وإيقاف المؤقت مؤقتًا بضغطة زر واحدة على العداد.
* **مؤشر الحالة:** يعرض الحالة الحالية بوضوح (RUNNING / PAUSE / DONE).
* **Three Time Modes:** Easily switch between Pomodoro (25 mins), Short Break (5 mins), and Long Break (15 mins).
* **Interactive UI:** A modern dark-themed interface featuring an SVG circular progress animation.
* **Smart Controls:** Click directly on the timer to toggle between Start and Pause.
* **Status Indicator:** Real-time text display showing the current state (RUNNING, PAUSE, or DONE).

---

##  التقنيات المستخدمة | Technologies Used

* **HTML5:** لبناء الهيكل البرمجي للصفحة وتوزيع العناصر (Semantic Elements).
* **CSS3:** لتصميم الواجهة العصرية، نظام الألوان الداكن، والتأثيرات الحركية (Custom Variables, Transitions, Flexbox).
* **JavaScript (ES6):** للتحكم في المنطق البرمجي للمؤقت، حساب الثواني، وتحديث الواجهة تلقائيًا (DOM Manipulation, SetInterval).

---

##  كيف يعمل المشروع؟ | How to Run

1. قم بعمل Clone للمستودع أو تحميل الملفات.
2. افتح ملف `index.html` في أي متصفح إنترنت.
3. اختر النظام المناسب ثم اضغط على الدائرة لبدء الإنتاجية!

1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
3. Select your desired mode and click on the circular timer to start!
