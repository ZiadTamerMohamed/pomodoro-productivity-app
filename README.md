# ⏱️ Pomodoro Timer Application

A sleek, elegant, and fully responsive Pomodoro Timer web application built from scratch using clean, modern, and native web technologies. This utility is engineered to help users optimize their workflows, manage time efficiently, and maximize productivity by seamlessly applying the globally acclaimed Pomodoro technique.

---

## Features

* **Three Integrated Time Modes:** Effortlessly toggle between focused work intervals (Pomodoro: 25 mins), short recovery periods (Short Break: 5 mins), and extended rest cycles (Long Break: 15 mins).
* **Interactive & Modern UI:** Features a dark-themed visual experience that reduces eye strain, equipped with a custom-styled circular timer interface.
* **Intelligent Controls:** Click directly on the timer element to intuitively switch between "Start" and "Pause" states without overloading the interface with unnecessary buttons.
* **Real-time Status Monitor:** Displays dynamic state changes explicitly tracking whether the timer is currently `RUNNING`, `PAUSE`, or `DONE`.

---

## Technologies Used

* **HTML5:** Semantic markup structure ensuring high accessibility, logical layout partitioning, and responsive element containers.
* **CSS3:** Custom UI design utilizing CSS Variable tokens for themes, flexible layouts via Flexbox, smooth transitions, and high-fidelity typography rendering.
* **JavaScript (ES6+):** Pure vanilla script handling precise interval tracking (`setInterval`), algorithmic second-to-minute conversions, and lightweight, performant DOM mutations.

---

## File Structure

* `index.html` - The static skeleton defining layout grids and essential interaction nodes.
* `style.css` - Custom style declarations, coloring schemes, variables, and responsive layout rules.
* `main.js` - Core application logic layer managing core calculations, timing execution loops, and DOM state updates.

---

## How to Run the Project Local

1. **Clone the repository** to your workspace or download the project archive.
2. Launch the `index.html` file inside **any modern web browser** (Chrome, Firefox, Edge, Safari).
3. Select your target productivity mode and click the timer circle to begin your deep-work session!
